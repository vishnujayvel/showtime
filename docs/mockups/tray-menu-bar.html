How It Works

Menu Bar Always Visible

Even when the pill is hidden or you're in another fullscreen app, the clapperboard icon sits in the macOS menu bar. A red tally dot pulses when you're live. One glance tells you your show status.

One-Click Actions

Jump to the expanded view, trigger Director Mode, or open the Writer's Room -- all from the dropdown. No need to find the pill or switch windows. The menu bar is your universal remote.

Replaces Pill for Minimal Users

Users who want zero screen clutter can hide the floating pill entirely and use only the menu bar. Timer, acts, beats, and all controls live in the dropdown. The clapperboard is your always-on control room.

Implementation Notes

Electron API

  • Tray class for menu bar icon
  • nativeImage.createFromPath() for icon states
  • Menu.buildFromTemplate() for dropdown
  • Use tray.setImage() to swap icon on state change

Design Decisions

  • Native Menu (not custom BrowserWindow) for reliability
  • Template images (@2x) for Retina + dark/light mode
  • Timer updates via tray.setTitle() next to icon
  • Rebuild menu on every showStore state change
