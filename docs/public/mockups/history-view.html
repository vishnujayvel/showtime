Design Notes

Verdict Badges

★ DAY WON 3/3 beats locked — Gold #f59e0b
SOLID SHOW 2/3 beats locked — Silver #9a9890
GOOD EFFORT 1/3 beats locked — Blue #60a5fa
SHOW CALLED EARLY 0/3 beats locked — Muted #5a5855

Interaction Notes

• Show rows have hover state (surface-hover #242428) with subtle transition

• Clicking a row navigates to the expanded detail view with a slide-up animation

• Week group headers appear when archive spans multiple weeks

• Summary bar at top provides at-a-glance stats and current streak

• Loading state uses shimmer animation matching skeleton dimensions of real content

• Empty state uses the "empty stage" metaphor — microphone on an empty stage

• Beat moments in detail view are optional — shown only if the user provided descriptions

• All transitions should use spring physics per the Showtime design system

Typography

• JetBrains Mono — Date numbers, verdict badges, stats, section labels, timestamps

• Inter — Act names, descriptions, body text, buttons
