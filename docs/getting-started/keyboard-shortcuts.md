# Keyboard Shortcuts

Quick reference for getting around Showtime without reaching for the mouse. All shortcuts use the macOS `Cmd` key.

## View Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd + 1` | Switch to Pill view (minimal floating timer) |
| `Cmd + 2` | Switch to Compact view |
| `Cmd + 3` | Switch to Dashboard view |
| `Cmd + 4` | Switch to Expanded view (full timer + lineup) |
| `Escape` | Collapse to Pill view |

## Show Controls

| Shortcut | Action |
|----------|--------|
| `Space` | Start or pause the current Act timer |
| `Cmd + Enter` | Lock a Beat during Beat Check |
| `Cmd + B` | Toggle Beat Check |
| `Cmd + I` | Start Intermission |
| `Cmd + D` | Open Director Mode |
| `Cmd + Shift + S` | Strike the Stage (end the show) |

## General

| Shortcut | Action |
|----------|--------|
| `Cmd + ,` | Open Settings |
| `Cmd + Q` | Quit Showtime |
| `Cmd + W` | Close window (collapses to Pill) |
| `Cmd + R` | Refresh (dev mode only) |

## Tips

- **Pill view is your home base.** When you're deep in work, collapse to Pill with `Escape`. The floating timer stays visible without taking up space.
- **Space to pause** works from any view. If you need to step away mid-Act, pause the clock. No penalty.
- **Director Mode (`Cmd + D`)** is always available. If the show is going off the rails, the Director is one shortcut away.
